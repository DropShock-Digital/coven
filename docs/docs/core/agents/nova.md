# Nova

**Nova** is Coven's default agent—your all-purpose familiar for general tasks, coordination, and daily work.

## What Nova Can Do

- Answer questions and research topics
- Help with writing and editing
- Coordinate with other agents
- Manage projects and tasks
- Provide advice and recommendations
- Remember your preferences and patterns

## Quick Start

```bash
# Chat with Nova
coven chat

# Ask a question
coven ask "What's the status of project X?"

# Delegate work
coven ask "Have Cody review the code and then Echo improve the docs"
```

## Personality

Nova is:
- **Helpful and practical** - Focuses on actionable outcomes
- **Curious** - Asks clarifying questions when needed
- **Adaptive** - Learns your working style
- **Honest** - Admits uncertainty rather than guessing

## Key Capabilities

- Web research
- Code understanding (not generation)
- Writing assistance
- Project coordination
- Memory management
- Delegation
