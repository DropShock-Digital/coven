# Welcome to Coven

**Coven** is a framework for building intelligent agents with persistent memory, multi-agent delegation, and seamless integration across platforms.

## What You Can Do

- **Build Agents** - Create autonomous agents with their own memory and capabilities
- **Delegate Work** - Agents can delegate tasks to other agents, creating hierarchical workflows
- **Persist Memory** - Agents maintain long-term memory across sessions
- **Integrate Anywhere** - Connect your agents to any platform via the gateway

## Quick Start

Get up and running in minutes:

```bash
# Install Coven
npm install -g @opencoven/cli

# Create your first agent
coven new my-agent

# Start the gateway
coven start
```

## Core Concepts

**Harness** - The runtime environment where agents execute  
**Familiar** - A persistent agent with its own memory and identity  
**Session** - A conversation or workflow between you and an agent  
**Memory** - Long-term storage of agent knowledge and context
